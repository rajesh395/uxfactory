CSS Introduction

CSS stands for cascading Style Sheet.It is a another language like html which is used to style an html document.It describes how Html elements are to be displayed on screen,paper or in other media.In this syntax has selecors and declaration.,in declaration again has property and value parts.

CSS has 3 ways to insert styles to an Html document.They are External CSS,Internal CSS,and Inline CSS.Basically External css means write styles in another file which is saved as CSS extension,we can use one External css fline in Multiple Html documents. Internal css means we can write styles in between STYLE Tag in Head part,it is appicable for only one document. Inline css means we can write styles within a html element in starting tag.
